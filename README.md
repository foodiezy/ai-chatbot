# 🐾 Companion Animals Chatbot

A multi-modal pedagogical chatbot designed for companion animals (Cats & Dogs). Integrates rule-based AI (AIML), Natural Language Processing (Similarity), First-Order Logic (Logical Reasoning), and Computer Vision (CNN).

## 🚀 Features
- **Conversational AI (AIML)**: Natural dialogue.
- **QA System (NLP)**: Similarity-based questions.
- **Logical Reasoning (FOL)**: Knowledge-base reasoning.
- **Image Classification (CNN)**: Animal identification.
- **Cloud Knowledge**: Wikipedia search integration.

## 🛠️ Prerequisites
- **Python 3.11**
- `pip install aiml requests nltk tensorflow pillow`

## 📖 How to Run
```bash
python chatbot.py
```
